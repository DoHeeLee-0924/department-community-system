웹 실행 방법

1. project_final 폴더에서 터미널 실행
2. 최초 1회 설치
   pip install -r requirements.txt
3. 웹 서버 실행
   python web_app.py
4. 브라우저 접속
   http://127.0.0.1:5000

관리자 계정
- 아이디: qwer
- 비밀번호: 1234

등급 기준
- 신입: 가입 직후. 인사방 조회/글쓰기 가능. 일반 게시글 상세 조회, 댓글, 좋아요, 신고 제한
- 일반: 게시글 1개 이상. 게시글 상세 조회, 댓글, 좋아요, 신고 가능
- 우수: 게시글 5개 이상 + 댓글 5개 이상. 신뢰도 점수 가중치 상승
- 최우수: 게시글 10개 이상 + 댓글 10개 이상. 신뢰도 점수 가중치 추가 상승
- 관리자: qwer/1234 고정 계정. 게시글/댓글 삭제, 신고 목록 조회 가능

주요 기능
- 태그 필수 선택: 인사방, 수업, 과제, 팀플, 공지, 취업
- 신입은 인사방 게시글만 상세 조회 가능
- 신입은 인사방에 글 1개 작성 후 일반 등급으로 자동 전환
- 일반 등급 이상부터 댓글, 좋아요, 게시글/댓글 신고 가능
- 신고 사유는 필수 입력
- 관리자는 신고 게시글/댓글을 한 화면에서 확인하고 삭제 가능

[추가 수정 기능]
- 작성자 본인은 본인이 작성한 게시글을 수정할 수 있습니다.
- 작성자 본인은 본인이 작성한 댓글을 수정할 수 있습니다.
- 관리자는 모든 게시글과 댓글을 삭제할 수 있지만 수정 권한은 없습니다.
